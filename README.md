# Six Animal Archive

A digital archive preserving every story, memory, and journey of Six Animal.

## Features

- SPA (Single Page Application) without frameworks
- Premium UI with Glassmorphism, Aurora, Neon, and Dark theme
- Smooth scroll, custom cursor, scroll progress, and mouse glow
- 3D particle background (Three.js)
- Member slider with Swiper.js
- Album Masonry with Lightbox (fullscreen, zoom, swipe, keyboard)
- Article search, filter, share
- Pustaka with preview & download
- FAQ accordion
- PWA ready with offline cache
- SEO optimized (Open Graph, Twitter Card, sitemap, robots)
- Lighthouse ≥ 95

## Tech Stack

- HTML5, CSS3, Vanilla JS (ES6)
- GSAP, Lenis, Swiper, AOS, Three.js

## Deployment

Deployed on Netlify.
